Markdown编辑器

习惯使用Sublime Text，一直都是在Sublime Text中装上Markdown插件使用。听说Typora很好用，一试~~~~

Typora

www.typora.io是一款支持实时预览的 Markdown 文本编辑器。也支持Linux平台，真的很不错。

代码块

作为一名程序员，先体验了下代码的展示功能。可以选择各种语言，我常用的C/C++/Python/Shell等都支持，默认的配色高亮简洁完美。

mermaid

sequence

选择Sequence语言即可。

李雷 -> 韩梅梅: Hello 梅梅, How are you?
Note right of 韩梅梅: 韩梅梅心想
韩梅梅 --> 李雷: I'm fine, thanks, and you?

李雷 -> 韩梅梅: Hello 梅梅, How are you?
